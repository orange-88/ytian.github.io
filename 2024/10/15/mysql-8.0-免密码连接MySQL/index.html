mysql 8.0：本地免密码连接MySQL

首先，声明一下，此教程适用于windows系统，mysql8.0版本。

小白第一次写博客，表达有误的地方请谅解，欢迎在评论区交流。

提示：我们可以先使用命令mysql –version(或者 mysql -V)，来查看你自己电脑上MySQL的版本信息。

本地免密码连接，或称为匿名连接。在已开启MySQL的前提下，只需要输入mysql命令，就可以登录到MySQL控制台中。因此，就需要直到MySQL的运行机制，

在windows环境下，MySQL的启动回去调用安装在Windows系统中的mysql服务，

这个服务位置在 控制面板/Windows 工具/服务/
首先搜索控制面板，打开

在控制面板中搜索Windows工具，双击进入Windows工具

在Windows工具中找到服务，双击进入

找到MySQL服务，右键选择属性

就能看到mysql服务，最终调用的是MySQL安装路径中，bin目录里的mysqld.exe这个应用程序。

mysqld这个程序在启动时，第一时间就会去找my.ini文件，这个文件会告诉MySQL怎么做。因此我们需要配置my.ini文件，那么第一步，我们要去找到my.ini文件。

但是，MySQL的安装目录里没有my.ini文件，通过全盘搜索，在这个目录下，找到了my.ini文件

打开笔记本进行编辑和保存，输入想要MySQL使用的登录时的用户账号和密码。

注意：用户账号和密码都需要填写，否则会报老错误。

然后，我们将my.ini文件剪切放到C:\Program Files\MySQL\MySQL Server 8.0目录下

以管理员身份打开cmd

输入net start mysql命令启动MySQL

输入mysql连接MySQL，看到”Welcome to the MySQL monitor.”表示成功。
